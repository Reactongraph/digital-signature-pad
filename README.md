# Digital Signature Pad (React + Canvas)

A fully standalone, dependency-free **digital signature pad** built with React and native HTML5 Canvas API. Supports drawing, role-based admin-client interaction, signature request workflow, clearing, and downloading.

---

## Features

- Draw smooth signatures with mouse or touch.
- Admin can request a client to sign.
- Client can complete the signature.
- Clear the canvas anytime.
- Download signature in **PNG**.
- Fully responsive and mobile-friendly.

## Admin View

- Click **Request Signature** to prompt the client to sign.

## Client View

- Draw your signature in the canvas.
- Click **Complete** to finish signing.

## Other Actions

- **Clear**: Clears the signature pad.
- **Download**: Save the signature as PNG or JPG.
- **Toggle User**: Switch between Admin and Client roles.
